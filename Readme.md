Building a Laundary Mart html Webpage 
----------------------------------------------------------------------------------------------------
It includes:

1.Header section: Welcoming heading using <h1></h1> tag.

2.About Section: Introducing the Laundary Mart. using <p></p> tag.

3.Image section : Image of Laundary : <img src=""> tag.

4.Services Section: introducing all the services provided using unordered list(ul).

5.Price Section: A tablr including all the prices of the services .(table tag with border).

6.Book Now section : Details of Customer using input tags.

7.Footer section: includes contact info of Laundary Mart.


## 🚀 How to Run & View

### Method 1: View the Live Deployment (Recommended)

If we want to view or test the source code locally:

2. Navigate to the project's root folder.
